2) Calcular a área e o perímetro de um retângulo
        
Área = base x altura
Perímetro = base + altura + base + altura

//Variáveis
area,base,altura,perimetro (Numéricas)
//Entrada
base, altura
//Processamento 1
area = base * altura
//Processamento 2
perimetro = base + altura + base + altura
//Saída 1
area
//Saída 2
perimetro

Calcular o volume de um aquário 

Multiplicar o comprimento pela largura, e em seguida pela altura. As medidas utilizadas no cálculo devem estar em centímetros. Com o valor obtido, deve-se dividir o resultado por 1.000.

//Variaveis
comprimento, altura, largura, volume (numéricas)

// Entrada
altura, largura, comprimento

//Processamento
volume = ((comprimento * largura)* altura)/1000

//Saida
volume
